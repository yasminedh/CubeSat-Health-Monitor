# CubeSat Health Monitor

CubeSat telemetry monitoring system using:

- ESP32 + FreeRTOS
- MQTT communication
- Flutter dashboard
- Firebase telemetry storage

## Features

- Real-time telemetry
- Safe mode simulation
- Battery monitoring
- Temperature monitoring
- Historical telemetry playback
- Firebase cloud storage

## Tech Stack

- Flutter
- Firebase
- ESP32
- FreeRTOS
- MQTT
